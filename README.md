# CAC Portal

Portal para el CAC (Club de Algoritmia de Cucei), el cual busca facilitar la toma de asistencia masiva y la generacion de reportes con base al desempe√±o y aprendizaje de los alumnos en las distintas actividades del club.

## Quick Start
``` bash
$ npm install
```

or

``` bash
$ yarn install
```

### Basic usage

``` bash
# dev server with hot reload at http://localhost:3000
$ npm start 
```

or 

``` bash
# dev server with hot reload at http://localhost:3000
$ yarn start
```

Navega a [http://localhost:3000](http://localhost:3000).

## Dependencias
No es una dependencia directa, pero el proyecto consume su propia API para realizar transacciones con la base de datos
- [CacAPI](https://github.com/IvanBM18/cac-api)
